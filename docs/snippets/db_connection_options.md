### `--host` (str)
IP address or DNS name of the database host.

- Default: `""` (local socket)
- Environment variable: `CETK_LT_DB_HOST`

### `--port` (int)
Port of the database.

- Default: `5432`
- Environment variable: `CETK_LT_DB_PORT`

### `--username` / `-u` (str)
Username for the database login.

- Default: `""`
- Environment variable: `CETK_LT_DB_USER_NAME`

### `--password` / `-p` (str)
Password for the database login.

- Default: `""`
- Environment variable: `CETK_LT_DB_USER_PW`
