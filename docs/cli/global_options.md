# Global Options

Global options must be placed **before** the subcommand.

```shell
cetk-lt [global option] [sub-command] ...
```

## Options

### `--help` / `-h`
Show help/usage details and exit.

### `--version`
Show `cetk-lt`'s version and exit.

---

### `--verbose` / `-v`
Show detailed debug messages on stderr (DEBUG level).

- Default: disabled
